As shown in the spectrum below, it seems the signal is filtered with band-pass filer, but the 50Hz artifact still exists. So I remove it here.

list of features¶

The function to extract features. The features are

  1. Integrated EMG
  2. Zero crossings
  3. Variance
  4. Slope sign changes
  5. Waveform length
  6. Kurtosis
  7. Skewness

Feature extraction is always the most important step, those parametric methods of course are not enough. If I have time, I will move on to the nonparametric method, such as dictionary learning, convolutional neural network, or similar classifier.

# input: Trial, \
#          sliding_window_start, \
#          IMFid,\
#          window_width,\
#          iirnotch_b,\
#          iirnotch_a
# output: X, you can image I have wrote good comments here
def extract_features_from_each_trial(Trial, \
                                     sliding_window_start, \
                                     IMFid,\
                                     window_width,\
                                     iirnotch_b,\
                                     iirnotch_a):

    Trial = signal.filtfilt(iirnotch_b,iirnotch_a,Trial)
    X = 0

    for win_s in sliding_window_start: 
        sliding_window = np.arange(win_s,win_s+window_width)
        data = Trial[sliding_window]

        emd = EMD() 
        IMFs = emd(data) 
        assembly = IMFs[IMFid].sum(axis=0)
        residual = data - assembly
        X_stack = np.vstack((data, IMFs[IMFid], residual))

        IEMG = np.abs(X_stack).mean(axis=1)
        ZC = (np.abs(np.diff(np.sign(X_stack)))/2).sum(axis=1)
        VAR = np.var(X_stack, axis=1)
        SSC = (np.abs(np.diff(np.sign( np.diff(X_stack) )))/2).sum(axis=1)
        WL = np.abs(np.diff(X_stack)).sum(axis=1)
        WAMP = np.abs(np.diff(X_stack)) 

        Kurtosis = scipy.stats.kurtosis(X_stack, axis=1)
        Skew = scipy.stats.skew(X_stack, axis=1)

        X_session = np.hstack((IEMG,ZC,VAR,SSC,WL,Kurtosis,Skew))
        X = X + X_session
#         X = np.hstack((X, X_session))
#         print(X.shape)

#     print(sliding_window_start.shape[0])
    X = X / sliding_window_start.shape[0]
    
    return X

# load saved features and their labels

# input: user,\
#          channel_name_list,\
#          col,\
#          normalization=False
# output: X_all, Y_all
def build_train_test_dataset(user,\
                             channel_name_list,\
                             col,\
                             normalization=False):
    X_all = np.empty(0)
    Y_all = np.empty(0)

    for cnt, channel_name in enumerate(channel_name_list):
        X_sub = np.empty(0)
        Y_sub = np.empty(0)
        for chID in [1,2]:
            X_subsub = np.load('features/Database1_'+user+'_'+channel_name+str(chID)+'.npy')

            if X_sub.size == 0:
                X_sub = X_subsub[:,col]
            else:
                X_sub = np.hstack((X_sub, X_subsub[:,col])) 

        if X_all.size == 0:
            X_all = X_sub
            Y_all = np.ones(X_sub.shape[0])*cnt
        else:
            X_all = np.vstack((X_all, X_sub)) 
            Y_all = np.hstack((Y_all, np.ones(X_sub.shape[0])*cnt))
            
    if normalization:
        X_all = X_all - X_all.mean(axis=0)
        X_all = X_all / X_all.std(axis=0)
    
    return X_all, Y_all

Train the classifier to detect grasp generalized to all users¶

This is actually an interesting question itself. The idea behind is to merge different user's feature, and make them look like they come from the same distribution. This is also related to state-or-the-art research about differential privacy. The idea is how can you remove the infomation associated with each user. My method is quite simple, just mean, std normalization.
